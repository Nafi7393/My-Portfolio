Random Art Generator

The Random Art Generator is a Python-based tool for creating abstract and patterned images programmatically. Customize lines, circles, rectangles, dots, colors, and effects to explore endless creative possibilities.

Explore the Project

Source code on GitHub:  Source on GitHub  |  Live demo:  Streamlit App

Features

  • - Image Generation Options: Horizontal/vertical lines, random lines, circles, rectangles, dots, padding and background control.

  • - Color Customization: Gradient start/end colors or fully randomized palettes.

  • - Effects & Transformations: Blending modes, mirror flips, and seamless tiling for pattern generation.

Installation & Usage

git clone https://github.com/Nafi7393/Random-Art-Generator.git
cd Random-Art-Generator
pip install -r requirements.txt

Run the generator script:
python art_generator.py


Specify options for image size, background color, number of elements, shapes (horizontal, vertical, random lines, circle, rect, dots), colors, radii, stroke, blending effect, mirroring, or seamless tiling.
