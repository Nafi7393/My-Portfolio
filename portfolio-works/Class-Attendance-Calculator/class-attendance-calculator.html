Class Attendance Calculator

Welcome to the Class Attendance Calculator, a Flask-based web application designed to simplify attendance management for classes. This project provides a user-friendly interface for both students and instructors to track attendance efficiently.

Explore the Project

You can access Class Attendance Calculator here: Live Demo  |  Source on GitHub

Features

  • - User-Friendly Registration and Login: Seamless signup and secure login to your ClassTrack account.

  • - Interactive Dashboard: View total classes attended, classes missed, and average attendance percentage at a glance.

  • - Visualize Dashboard: Interactive charts powered by Chart.js to track attendance trends over time.

  • - Course Management: Easily add, edit, or remove courses from your dashboard.

  • - Real-time Updates: Attendance records update instantly as you make changes.

  • - Responsive Design: Fully optimized for desktop, tablet, and mobile devices.

  • - Privacy and Security: State-of-the-art encryption and best practices to protect your data.

Technologies Used

  • - Front-end: HTML, CSS, JavaScript

  • - Back-end: Python (Flask)

  • - Database: PostgreSQL

  • - Charts: Chart.js

  • - Session Management: Flask-Session

Installation


git clone https://github.com/Nafi7393/Class-Attendance-Calculator.git
cd Class-Attendance-Calculator
pip install -r requirements.txt

Optional: Set your Flask environment:
export FLASK_APP=main.py
export FLASK_ENV=development

Start the server with:

flask run

Then open http://localhost:5000 in your browser.

Usage

  1. - Register or log in to your Class Attendance Calculator account.
  2. - Add your courses and mark attendance as needed.
  3. - Monitor your statistics and charts on the dashboard.
  4. - Update records anytime and see real-time changes.
  5. - Log out when finished to secure your session.
