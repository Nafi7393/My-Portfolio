Pinterest Post Maker

This Python script generates Pinterest‐style post images by overlaying text on background images. It utilizes threading for concurrent image generation and supports customization of text styling.

Explore the Project

Source code available on GitHub: Source on GitHub

Requirements

  • Python 3.x

  • PIL (Python Imaging Library)

  • `impact.ttf` font file (or specify your own font)

  • Background images in the background_images folder

  • Input text CSV (`pin_text.csv`)

Installation

git clone https://github.com/Nafi7393/PINTEREST-POST-MAKER.git
cd PINTEREST-POST-MAKER

pip install pillow

Usage

Modify pin_text.csv with your desired text lines. Then run: python main.py

You can adjust parameters in main.py, such as max_limit in the fire_pins function for batch generation.

Example Images

Customization

  • Text Styling: Modify text_color, font_path, base_font_size, and font_increase_size in the PinMaker class.

  • Backgrounds: Add or change images in the background_images folder.

Notes

Ensure the CSV file (`pin_text.csv`) is correctly formatted with each line of text in the first column. The script uses threading for concurrent processing, enabling efficient batch creation.
