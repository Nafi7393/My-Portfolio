Awesome QR Codes Generator

Welcome to the Awesome QR Codes Generator, a Streamlit-based application that simplifies the creation of customized QR codes. Enter text, upload background images, pick your colors, and download instantly.

Explore the Project

Source code available on GitHub: Source on GitHub

Features

  • - Customization Options: Input text, upload a background image, and choose QR & background colors.

  • - Instant Download: Download your generated QR codes directly from the interface.

  • - Example imgs: Browse creatively customized QR code examples in the imgs directory.

Installation

git clone https://github.com/Nafi7393/Awesome-QR-Codes-Generator.git
cd Awesome-QR-Codes-Generator
pip install -r requirements.txt

streamlit run main.py

Usage

  1. - Input the text you want to encode.
  2. - Upload an optional background image.
  3. - Choose QR code and background colors.
  4. - Click “Generate Now” to create your QR code.
  5. - Download your QR code with one click.
