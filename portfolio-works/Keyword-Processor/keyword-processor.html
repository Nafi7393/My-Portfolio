Keyword Processor

The Keyword Processor is a Streamlit application designed to extract keywords, fetch synonyms and antonyms, display definitions, and visualize relationships using dynamic word clouds. It leverages NLP techniques for an interactive linguistic exploration.

Explore the Project

Source code available on GitHub: Source on GitHub

Features

  • - Keyword Extraction: Uses the RAKE algorithm to extract significant keywords from input text.

  • - Synonyms & Antonyms: Fetches synonyms and antonyms for each keyword via external APIs.

  • - Definitions: Retrieves detailed definitions from a dictionary API.

  • - Word Cloud Visualization: Generates dynamic word clouds for synonyms and antonyms.

  • - Interactive Interface: Streamlit-based UI with real-time updates and buttons.

Usage

  1. Clone the repository:
    git clone https://github.com/Nafi7393/Keyword-Processor.git
    cd Keyword-Processor
  2. Install dependencies:
    pip install -r requirements.txt
  3. Launch the Streamlit app:
    streamlit run main.py
  4. Open the provided URL in your web browser to access the application.
