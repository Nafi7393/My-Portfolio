Midjourney Prompt Generator

The Midjourney Prompt Generator is a Streamlit application that helps users craft high-quality prompts for Midjourney by integrating with the Gemini API. Simply enter your base idea, style preferences, mood, and more—and get multiple refined prompts ready to feed into Midjourney.

Explore the Project

Source code available on GitHub: Source on GitHub

Features

  • - User-Friendly Interface: Streamlit web app for easy input of prompt parameters.

  • - Customizable Prompts: Specify base idea, art style, mood, color palette, and more.

  • - API Integration: Generates refined prompts via the Gemini API.

  • - Prompt Management: Displays multiple generated prompts ready for Midjourney.

Getting Started

Prerequisites
  • Python 3.7+
  • Streamlit
  • python-dotenv (for environment variables)
  • google-generativeai (Gemini API client)
Installation
  1. Clone the repo:
    git clone https://github.com/Nafi7393/Midjourney-Prompt-Builder.git
    cd Midjourney-Prompt-Builder
  2. (Optional) Create & activate a virtual environment:
    python -m venv venv
    source venv/bin/activate (or venv\Scripts\activate on Windows)
  3. Install dependencies:
    pip install -r requirements.txt
  4. Create a .env file with your Gemini API key:
    GEMINI_API_KEY=your_gemini_api_key

Usage

  1. Launch the Streamlit app:
    streamlit run app.py
  2. Open http://localhost:8501 in your browser.
  3. Fill in your base idea, style, mood, elements, and other details.
  4. Click Generate Prompts to see your tailored Midjourney prompts.
